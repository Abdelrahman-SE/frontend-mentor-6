# Four Card Feature Section

A responsive **Four Card Feature Section** built using **HTML5** and **CSS Grid**, inspired by the Frontend Mentor challenge.  
The project focuses on clean layout structure, responsive design, and modern CSS practices.

## Technologies Used

- **HTML5**
- **CSS3**
- **CSS Grid**
- **Flexbox**
- **Responsive Design**
- **CSS Custom Properties (Variables)**

## Features

- Fully responsive layout for mobile, tablet, and desktop screens
- Clean and modern card-based UI
- CSS Grid used for complex layout positioning
- Custom CSS variables for colors and typography
- Smooth and readable typography using Google Fonts (Poppins)
- Well-structured and maintainable code
